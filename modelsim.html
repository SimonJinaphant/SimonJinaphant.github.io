Why OpenGL

The OpenGL API provides high performance manipulation of computer graphics; it harness the physical Graphical Processing Unit (GPU) to deliver computational power for repetitive and demanding tasks in which the CPU wasn't designed to handle effectively.

The modern pipeline OpenGL uses allows software developers to have more control on how an image is rendered through the usage of shaders. The pipeline operates in sequential manner, starting from the vertices transformations via the vertex shader, to determining how each pixel should be coloured and blended after the fragment shader.

The OpenGL API is also popular as it has been ported to many different types of platforms; it can be used for embedded systems such as mobile phones or tablets via OpenGL-ES, or on web based systems with WebGL. It should be noted that these variations do not offer a complete coverage of the original API but still contains the fundamental packages.
